# MongoDbProject
### MongoDb ile Farm Projesi
###### Api katmanında oluşturduğum Ürünler ve Kategorileri Mongo Db ile ayağa kaldırdım, UI katmanında projeye api consume ile çektim.
##### Kullanılan Teknolojiler
*.Net Core 6.0* <br/>
*MongoDb* <br/>
*Auto Mapper* <br/>
